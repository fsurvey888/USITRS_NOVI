# Водич за приступ Админ панелу

## 🔒 Скривени админ панел

Админ панел је **скривен из главне навигације** и не видљив на веб-сајту. Ово чини систем безбеднијим јер администратори могу приступити без експозиције линка јавности.

## 📍 Како приступити админ панелу?

### Метод 1: Директан приступ преко URL-а (препоручено)

1. Отвори браузер
2. Унеси следећу URL адресу у адресну линију:
   ```
   https://твој-сајт.netlify.app/admin-login
   ```

3. Или локално приликом развоја:
   ```
   http://localhost:3000/admin-login
   ```

### Метод 2: Коришћење закалдника (Bookmark)

1. Отвори `/admin-login` страну
2. Сачувај страницу као закладник у браузеру
3. Следећи пут, само кликни на закладник да се брзо вратиш на login страну

### Метод 3: Скраћена адреса

Можеш направити скраћену адресу помоћу:
- **Bit.ly** или сличног сервиса
- **QR код генератора** (за брз приступ на мобилном уређају)

## 🔑 Логирање

На login странице унесите:

| Поље | Вредност |
|-------|----------|
| **Корисничко име** | `predsjednika` |
| **Лозинка** | `usit2025` |

> ⚠️ **Сачувајте креденцијале на безбедном месту!**

## 📊 Администраторске функције

После логирања, имате приступ:

### 1. **Вијести** (Картица "Додај вијест")
   - Додај нову вијест
   - Унесите: наслов, опис, датум, садржај
   - Додајте слику (URL)
   - Додајте галерију слика (раздвојене зарезом)
   - Поставите почетни број прегледа
   - Бројач прегледа расте аутоматски при посећивању

### 2. **Занимљивости** (Картица "Занимљивости")
   - Додај занимљивост
   - Унесите: наслов, занимљивост, датум
   - Додајте слику (thumbnail)
   - Изаберите икону

### 3. **Документи** (Картица "Документи")
   - Додај документ
   - Унесите назив и линк (Google Drive, Dropbox, итд)
   - Откупите категорију

### 4. **Управљање** (Картица "Управљање")
   - Видите све додане ставке
   - Редактујте садржај
   - Обришите ставке

## 🖼️ Додавање слика

### Слике за вијести и занимљивости

1. **Опција 1: Веб URL**
   - Прилепите URL слике (https://example.com/slika.jpg)
   - Слика мора бити јавно доступна

2. **Опција 2: GitHub складиште**
   - Додајте слику у `/public/images/` фолдер
   - Commit и push на GitHub
   - Коришћење: `/images/име-слике.jpg`

3. **Опција 3: Облачно складиште**
   - Google Drive
   - Dropbox
   - OneDrive
   - Инстаграм или друге платформе

## 🔄 Галерија слика

За галерију вијести:
- Унесите више URL-а раздвојених зарезом
- Пример:
  ```
  https://slika1.jpg, https://slika2.jpg, https://slika3.jpg
  ```

## 📈 Праћење прегледа

Бројач прегледа:
- Почиње од вредности коју унесете приликом креирања
- Расте за 1 сваки пут када читалац посети вијест
- Максимално једном по часу по браузеру (не броји исти браузер вишепута за кратко време)

## 🛡️ Безбедност

### Заштита административних креденцијала

1. **НИКАД не делите URL за login јавно**
2. **Чувајте пароле безбедно** - користите лозинчник (Bitwarden, LastPass, итд)
3. **Редовно мењајте пароле** - препоручује се сваких 3 месеца
4. **Логујте се из браузера** - кликните "Одјави се" када завршите
5. **Не делите экран** приликом логирања

### Променa лозинке

За промену лозинке, требате:
1. Отворите `lib/admin-config.ts` фајл
2. Промените вредност у `"usit2025"` на нову лозинку
3. Commit и push на GitHub
4. Netlify ће аутоматски редеплој

> Препоручује се коришћење јаке лозинке (мешовина слова, бројева, специјалних карактера)

## 🚀 Редигирање контента

### Редактовање постојеће вијести

1. Идите на картицу "Управљање"
2. Нађите вијест коју желите да уредите
3. Кликните "Уреди"
4. Направите измене
5. Кликните "Сачувај"

### Брисање контента

1. Идите на картицу "Управљање"
2. Кликните "Обриши" поред ставке
3. Потврдите брисање
